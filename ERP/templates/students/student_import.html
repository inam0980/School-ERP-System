How it works / كيف يعمل

  • Export the student list as CSV from the Students page.
  • Edit the file if needed — do not rename any column headers.
  • Upload the same CSV file here.
  • If Student ID matches an existing record → the student is updated.
  • If Student ID is blank or not found → a new student is created.
